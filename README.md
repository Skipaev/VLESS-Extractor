# VLESS-Extractor

Простой и мощный инструмент на Python для извлечения чистых VLESS-ссылок из подписок (subscription links).

### Для чего нужен
- Получает подписку по URL
- Отправляет HWID и данные устройства (как это делает клиент Throne)
- Поддерживает маскировку под Throne (User-Agent, заголовки x-hwid, x-ver-os и т.д.)
- Декодирует base64, парсит sing-box JSON, vmess → vless конвертация
- Выдаёт чистые vless:// ссылки, готовые для импорта в любой клиент (Nekobox, v2rayNG, Hiddify и т.д.)
- Показывает Subscription-Userinfo (трафик, expire)

### Основные возможности
- Полная совместимость с механикой Throne по запросу подписки
- Поддержка кастомных HWID-параметров
- Игнор SSL-ошибок (опционально)
- Парсинг sing-box outbounds (vless + vmess)
- Удобный вывод и меню (показать, сохранить, скопировать, base64)
